# Project Requirements

## Overview

Personal portfolio site for Karina Macancela, software engineer. Showcases work, experience, skills, and blog posts. Built with React, Tailwind CSS, and TypeScript.

## Core Requirements

- Multi-page architecture with distinct routes for each section
- Responsive design across all screen sizes
- Performant animations that respect reduced-motion preferences
- Earthy/watery/artsy aesthetic — no dark/neon/techy vibes
- Fast load times and good Lighthouse scores
- Deployed on Vercel

## Pages (Planned)

- **Home** — Hero + brief highlights, CTAs to other pages
- **About** — Bio, skills, personal details
- **Projects** — Project showcase with detail views
- **Experience** — Work history timeline
- **Blog** — Blog listing and individual post pages
- **Contact** — Contact form or info

## Technical Requirements

- TypeScript strict mode
- Tailwind CSS v4 with design tokens (no custom values)
- React Router v7 for client-side routing
- Motion library for animations
- Static data in `src/data/` (no backend)
- SEO-friendly meta tags per page

## Out of Scope (V1)

- CMS or backend API
- Authentication
- Comments system
- Analytics dashboard
